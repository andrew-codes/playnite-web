# Suggested Commands

## Testing
- `yarn nx test/unit <project>`: Run unit tests
- `yarn nx test/component <project>`: Run component tests
- `yarn nx test/e2e <project>`: Run e2e tests

## Development
- `yarn nx start web`: Start Playnite Web application
- `yarn nx build <project>`: Build project
- `yarn nx lint <project>`: Run linter

## Package Management
- `yarn`: Install all workspace dependencies (from repo root)
- Do NOT use npm

## System (Darwin/macOS)
- Standard Unix commands: ls, cd, grep, find, git
